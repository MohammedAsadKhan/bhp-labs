# bhp-labs

> Security tool implementations from *Black Hat Python, 2nd Edition* by Justin Seitz & Tim Arnold.  
> Built for learning purposes — all tools are for **authorized use only** in controlled/lab environments.

---

## About

This repo documents my journey through *Black Hat Python* — each chapter gets its own folder with working implementations, personal notes, and any modifications or improvements I made along the way.

---

## Structure

```
bhp-labs/
├── ch02-networking/         # Netcat replacement, SSH tools
├── ch03-network-offense/    # Packet sniffing, ARP poisoning
├── ch04-owning-the-network/ # Burp proxy, web recon
├── ch05-web-hackery/        # Directory busting, HTML parsing
├── ch06-extending-burp/     # Burp Suite extensions
├── ch07-github-c2/          # Command & control via GitHub
├── ch08-trojans/            # Trojan framework
├── ch09-exfiltration/       # Data exfiltration techniques
├── ch10-windows-privesc/    # Windows privilege escalation
└── ch11-forensics/          # Forensics & detection
```

Each chapter folder contains its own `README.md` with tool descriptions, usage, and notes.

---

## Disclaimer

All tools in this repository are built **strictly for educational purposes**. Only run these against systems you own or have explicit written permission to test. Unauthorized use against systems you do not own is illegal.
